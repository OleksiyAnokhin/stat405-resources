Command line-setup

Linux

In the terminal, run the following code to install the command-line settings that you need:

curl http://had.co.nz/stat405/resources/linux-dotfiles.zip -o dotfiles.zip
unzip dotfiles.zip
rm dotfiles.zip

Close and re-open the terminal.

Mac

Before run the following code, check to see if .profile already exists by running ls -a .profile. If it does, see me.

In the terminal, run the following code to install the command-line settings that you need:

curl http://had.co.nz/stat405/resources/mac-dotfiles.zip -o dotfiles.zip
unzip dotfiles.zip
rm dotfiles.zip
