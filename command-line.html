Linux

In the terminal, run the following code to install all the command-line settings that you need:

curl http://had.co.nz/stat405/resources/linux-dotfiles.zip -o dotfiles.zip
unzip dotfiles.zip
rm dotfiles.zip

Close and re-open the terminal.
