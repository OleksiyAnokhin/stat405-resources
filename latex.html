Latex

A few of you have asked me how you can use latex on your own computers. This page describes (to the best of my knowledge) the best tools to use on Windows and Mac. With these tools you won't need to use the command line, but try and stick with the concept of the working directory, with all related files living in the same directory.

If you're a PhD student in statistics, I'd suggest you spend some time getting familiar with these tools as you will be using them a lot.

Windows

  • Latex: miktex (~600 meg)
  • Text editor: winedt
  • Using miktex and winedt together (this means you don't need to run anything from the command line).

Mac

  • Latex: mactex (1.15 gig)
  • Text editor: TeXShop, included in mactex.
