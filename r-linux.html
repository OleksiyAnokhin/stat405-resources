Using R on linux

Installation

R is already installed on the lab machines. To install R on your own computer, you can use your usual package management software to install R. See the linux pages on the R site for more information.

Text editors

For linux, I recommend that you use gedit, which can be opened from the command line with gedit filename &.

To configure it for maximum compatibility with our R style, open Edit | Preferences and then make sure the settings match those in this pdf.

Keyboard shortcuts

  • Switch applications: alt + tab
  • Start of line: ctrl + a
  • End of line: ctrl + e
  • Copy: ctrl + shift + c
  • Paste: ctrl + shift + v

Working directories

Start R in the directory in which you wish to work.
