Linux setup

After working through this page, your account will be set up for everything we need for the rest of the semester. You should only need to do this once.

gedit settings

Open gedit by typing gedit & in the terminal. Open Edit | Preferences and then make sure the settings match those in this pdf.

Command line & R settings

Run gedit .cshrc & and copy in the following code:

test -t 0 && exec /bin/bash

Save and close the file.

Run gedit .bashrc & and copy in the following code:

export R_LIBS=$HOME/R
alias R='R --no-save --no-restore-data --quiet'

export INPUTRC=~/.inputrc 
export EDITOR="gedit" 

# Save lots of history 
export HISTFILESIZE=10000 
export HISTSIZE=10000 
export HISTCONTROL=erasedups 
shopt -s histappend 

# Show which directory you're working in 
export PS1="\W: "

Save and close the file.

Run gedit .Rprofile & and copy in the following code:

.First <- function() {
  options("repos" = c(CRAN = "http://cran.r-project.org/"))
}

Run gedit .inputrc & and copy in the following code:

Save and close the file.

set completion-ignore-case on 
$if Bash 
  set show-all-if-ambiguous on 
$endif 

# Search backwards and forwards in history 
# using up & down keys 
"\e[A": history-search-backward 
"\e[B": history-search-forward

Save and close the file.

Close and re-open the terminal.
