Using R on the mac

Installation

To download R for Mac OSX 10.4 and higher:

  1. Visit http://cran.r-project.org
  2. Click on the "Mac OSX" link
  3. Download the first file (something.pkg)
  4. Open the file and install R

Text editors

R comes with a built in text editor. When running R through the command line, however, you'll want an external text editor. For Macs, I recommend:

  • Free: fraise
  • Pay: textmate

Keyboard shortcuts

  • Switch applications: command + tab
  • Copy: command + c
  • Paste: command + v
  • Cut: command + x
  • Undo: command + z
  • Redo: command + shift + z
  • Run highlighted line (when using R text editor): command + return

Working directories

From R.app, press command + d. From the command line, it's the directory you start R from.
