Using R on windows

Text editors

R comes with a built in text editor, but you might want a more fully featured text editor.

I'd recommend Winedt, which costs $40 (cheaper than a textbook!), because it integrates well with latex and R. install RWinEdit, by running install.packages("RWinEdt") within R. When you want to edit R files, run library(RWinEdt) in R. You might also want to read an introduction to RWinEdit.
