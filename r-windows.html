Using R on windows

Installation

  1. Visit http://cran.r-project.org
  2. Click on the "Windows" link
  3. Click on the "base" link
  4. Click "Download R 2.11.1 for Windows" (or whatever version is displayed at the top of the page)
  5. Open the file and install R

Text editors

R comes with a built in text editor, but you might want a more fully featured text editor.

I'd recommend Winedt, which costs $30 (cheaper than a textbook!), because it integrates well with latex and R. install RWinEdit, by running install.packages("RWinEdt") within R. When you want to edit R files, run library(RWinEdt) in R. You might also want to read an introduction to RWinEdit.

You can also try the free eclipse with R and latex plugins.

Keyboard shortcuts

  • Switch applications: alt + tab
  • Start of line: ctrl + a
  • End of line: ctrl + e
  • Copy: ctrl + c
  • Paste: ctrl + v
  • Undo: ctrl + z
  • Run highlighted line (when using R text editor): F5

Working directories

Set your working directory by clicking File | Change dir.
