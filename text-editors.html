Text editors

My text editor is the one tool that I use the most. I use it for writing R code, latex, web pages, ruby scripts and for taking notes. Text files have the great advantage that you can always read them - you never have to rely on having purchased some software.

A small amount of effort learning the features of your text editor can provide great dividends. I would suggest that you try and figure out if your text editor can do each of the following tasks, and what the keyboard shortcut is:

  • Bulk comment the selected lines
  • Indent/dedent
  • Send the current line or selected text to R

Linux

Open linux, I recommend that you use gedit, which can be opened from the command line with gedit filename &.

To configure it for maximum compatibility with our R style, open Edit | Preferences and then make sure the settings match those in this pdf.
