Latex on linux

Installation

It's already done for you :)

Edit

Open your latex files in gedit, e.g. gedit template.tex &.

Compile

In the terminal, navigate to the directory where the latex file lives. Run pdflatex your-file-name.tex. Latex will output progress to the terminal.

Unfortunately it's really hard to see any warnings or errors, because they are buried in rafts of uninformative material. If you pdf is not built successfully, the best thing is to open the log file in gedit (gedit your-file-name.log) and search for "error".

Preview

If successful, the compilation step will have created a pdf called your-file-name.pdf. Preview it by running evince your-file-name.pdf
