Using R

Mac OSX

Text editors

R comes with a built in text editor. When running R through the command line, however, you'll want an external text editor. For Macs, I recommend:

  • Free: smultron
  • Pay: textmate

Keyboard shortcuts

  • Copy: command + c
  • Paste: command + v
  • Cut: command + x
  • Undo: command + z
  • Redo: command + shift + z

Download location

To download R for Mac OSX 10.4 and higher, visit http://cran.r-project.org and follow the "Mac OSX" link. Download R-2.92.dmg (latest version). Open the file and install R.
